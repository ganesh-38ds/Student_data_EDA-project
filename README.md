# Student Performance Exploratory Data Analysis (EDA)

## Objective

Analyze student performance data to identify patterns, trends, and factors affecting academic success.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Analysis Performed

* Dataset Overview
* Missing Value Analysis
* Gender Distribution
* Age Distribution
* Study Time Analysis
* Absence Analysis
* Failure Analysis
* Correlation Analysis
* Outlier Detection

## Key Findings

* Study time positively impacts final grades.
* Absences negatively affect performance.
* Previous grades strongly correlate with final grades.
* Students with failures generally achieve lower scores.

## Conclusion

EDA helped uncover the major factors influencing student academic performance and provided valuable insights into learning outcomes.
